El presente repositorio contiene un código para automatizar el sistema de inventario de la empresa "Meilí", considerando que la misma realizaba esta operación de manera manual.
El repositorio contiene el código python utilizado para el proceso, así como los archivos .xlsx utilizados para probar las operaciones del mismo.
El código Python permite tanto el acceso a un documento .xlsx previamente creado, como la generación de uno nuevo. 
Así mismo, también permite la edición de los datos presentes en la tabla de datos.
Entre las ediciones posibles, se encuentran:
1. Guardado de cambios en el archivo.
2. Cálculo automático de las cantidades en Stock
3. Añadir un nuevo producto
4. Eliminar un producto
5. Añadir compra de productos ya existentes
6. Añadir venta de productos ya existentes
7. Añadir elementos de productos ya existentes que se han dañado

También, se tiene una funcionalidad básica de interfaz de usuario (Se planea expandir esta funcionalidad a futuro, con uso de elementos html, javascript y demás).
